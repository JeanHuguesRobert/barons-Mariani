# Collections — Musée Mariani

Ce dossier indexera les objets, lieux, documents, images et ensembles patrimoniaux susceptibles d'être intégrés au Musée Mariani.

## Catégories de collection

| Catégorie | Description | Statut |
|---|---|---|
| Objets | mobilier, objets familiaux, objets symboliques | à inventorier |
| Documents | actes, lettres, photographies, plans, archives | à inventorier |
| Lieux | Minesteggio, jardins, hydraulique, bâtiments, parcelles | à documenter |
| Personnes | figures familiales, alliés, témoins, acteurs historiques | à relier aux dossiers |
| Récits | traditions familiales, épisodes locaux, hypothèses | à qualifier |
| Uchronies | scénarios contraints de médiation historique | à séparer des faits établis |

## Modèle de fiche objet

```text
# Nom de l'objet

## Description courte
## Localisation actuelle
## Propriétaire / détenteur
## Datation
## État matériel
## Histoire connue
## Sources associées
## Incertitudes
## Usage muséal possible
## Droits et restrictions
## Images disponibles
```

## Règle

Aucun objet ne doit être réduit à sa valeur décorative. Chaque objet doit être relié à une preuve, un récit, une question historique ou une fonction de transmission.
