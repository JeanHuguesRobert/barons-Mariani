# Preuves et incertitudes — méthode critique du Musée Mariani

Le Musée Mariani doit rendre visibles les degrés de preuve. Cette règle est essentielle pour éviter deux erreurs symétriques : transformer une tradition fragile en certitude, ou effacer un indice intéressant parce qu'il n'est pas encore pleinement démontré.

## 1. Grille de qualification

| Niveau | Qualification | Usage muséal |
|---:|---|---|
| 1 | Fait établi | peut être affirmé sans réserve excessive, avec source |
| 2 | Fait probable | peut être utilisé avec prudence, en indiquant la source et les limites |
| 3 | Tradition familiale ou généalogique | peut être conservée comme indice, non comme preuve complète |
| 4 | Hypothèse historique | peut guider l'enquête, sans être présentée comme fait |
| 5 | Interprétation | peut structurer le récit, si elle reste distinguée des faits |
| 6 | Uchronie contrainte | peut être exposée comme simulation, jamais comme histoire advenue |
| 7 | Fiction libre | hors du noyau probatoire ; usage éventuel seulement dans une section explicitement fictionnelle |

## 2. Formulations recommandées

| Cas | Formulation faible | Formulation correcte |
|---|---|---|
| Source absente | « il est certain que » | « une tradition indique que » |
| Titre ambigu | « il était prince » | « il aurait reçu un titre ou nom honorifique » |
| Hypothèse | « cela prouve que » | « cela suggère une piste à vérifier » |
| Uchronie | « Napoléon a vécu à Minesteggio » | « le scénario uchronique explore ce qui changerait si... » |
| Généalogie secondaire | « la filiation est prouvée » | « la filiation est donnée par telle source, à contrôler par actes » |

## 3. Règle de conservation

Une information fragile ne doit pas être supprimée par prudence excessive. Elle doit être :

1. conservée ;
2. datée ;
3. sourcée si possible ;
4. qualifiée ;
5. reliée à une liste de vérifications ;
6. séparée des faits établis.

## 4. Règle de présentation publique

Le public doit pouvoir comprendre le degré de certitude sans être noyé dans l'appareil critique.

Modèle :

> Cette mention est attestée par une tradition généalogique, mais aucune source primaire n'a encore été identifiée. Elle est donc présentée comme un indice d'enquête, non comme une certitude.

## 5. Règle coloniale

Les traces coloniales doivent être traitées avec une double prudence :

- ne pas les exploiter comme trophées familiaux ;
- ne pas les effacer lorsqu'elles permettent de comprendre les rapports de pouvoir.

Le dossier Louis-Thomas Mariani / Tahiti est le premier cas d'application de cette règle.

## 6. Règle uchronique

Le Musée uchronique peut explorer des scénarios contrefactuels, mais chaque scénario doit être explicitement séparé de l'histoire advenue.

L'uchronie acceptable est une **fiction contrainte** : elle part d'un point de bifurcation identifiable et respecte les contraintes historiques, sociales, matérielles et territoriales.

## 7. Règle source / produit

Le dépôt source conserve les preuves, variantes et incertitudes. Le produit muséal en donne une version lisible.

Formule :

> Le cartel simplifie ; le corpus justifie.
