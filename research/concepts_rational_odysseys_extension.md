## The Possible

**Type:** philosophical / potentic / modal framework  
**Scope:** Global  
**Status:** Working

**Short definition:**  
The total domain of what reality has not forbidden: what may become actual under some configuration of physical, logical, technical, social, institutional, economic, cognitive, and temporal constraints. Distinguished from local *possibilities*, which are only the visible or situated options a given actor perceives at a given time.

**Core formula:**  
Possibilities are what a situation seems to offer. The Possible is what reality has not forbidden.

**Parent concepts:**
- Potentics
- Possibilism

**Child concepts:**
- Actualized possible
- Latent possible
- Blocked possible
- Potentiality
- Rational Odyssey

**Related concepts:**
- Machine à explorer
- Rational exploration
- Spirit of Synthesis
- The Possible versus possibilities
- Autonomie de capacité

## Rational Odyssey

**Type:** public explanatory grammar / mission model  
**Scope:** Global  
**Status:** Seed / Working

**Short definition:**  
A mission-oriented exploration of The Possible structured as departure, exploration, trace leaving, ordeal, return, map update, and transmission. The Odyssey is used as a public grammar for the rational exploration of The Possible: a mission starts from an Ithaca, crosses uncertain terrain, leaves traces, avoids capture, returns to the corpus, and sometimes produces a theory.

**Core formula:**  
Organize rational Odysseys into The Possible.

**Parent concepts:**
- Potentics
- Machine à explorer

**Child concepts:**

- Ariadne thread / continuation
- Rich stigmergy

- Return protocol

## Ithaca / Return to Corpus

**Type:** operational metaphor / corpus protocol  
**Scope:** Global  
**Status:** Seed

**Short definition:**  
The symbolic and operational return point of a mission. Ithaca names the place where exploration stops being private experience and becomes transmissible memory: debrief, trace sorting, artifact promotion, map update, open questions, and possible theory candidate.

**Core formula:**  
Every mission needs its Ithaca: the place where experience returns and becomes transmissible memory.

**Parent concepts:**
- Rational Odyssey
- Corpus

**Related concepts:**
- FractaLog
- Return protocol
- Mission
- Artifact promotion

## Ariadne Thread / Continuation

**Type:** operational metaphor / continuation model  
**Scope:** COP / Cogentia / FractaLog  
**Status:** Seed / Working

**Short definition:**  
The minimum trace that prevents exploration from becoming imprisonment. In COP, Ariadne's thread maps to continuations, provenance, checkpoints, FractaLog entries, and return paths that allow a suspended exploration to resume or return safely.

**Core formula:**  
Ariadne's thread is the minimum trace that prevents exploration from becoming imprisonment.

**Parent concepts:**
- Rational Odyssey
- Continuation Protocol

**Related concepts:**
- COP Mission
- FractaLog

## Rich Stigmergy

**Type:** coordination model / agentic infrastructure pattern  
**Scope:** Global / COP / Cogentia  
**Status:** Working

**Short definition:**  
A method of ultra-distributed coordination through traces left by agents in a shared environment. Unlike simple insect-like stigmergy, rich stigmergy uses structured cognitive traces: promising paths, dead ends, warnings, verified claims, weak hypotheses, contradictions, return requirements, synthesis candidates, confidence levels, evidence references, and decay policies.

**Core formula:**  
Stigmergy is the minimal memory left in the world so that the next intelligence does not start again from zero.

**Parent concepts:**

## Flotilla Architecture

**Type:** distributed architecture pattern  
**Scope:** Agentic systems / COP / Fractanet  
**Status:** Seed

**Short definition:**  
A distributed exploration architecture based on many small, partially autonomous agents rather than a single massive central agent. It favors resilience, loss tolerance, parallel exploration, and recomposition after disruption, provided that protocols exist for dispersion, rendezvous, signaling, trace merging, and return.

**Core formula:**  
A flotilla is superior to the Titanic only if it has protocols for dispersion, rendezvous, signaling, recomposition, and return.

**Parent concepts:**
- Machine à explorer
- Fractanet / COP

**Related concepts:**
- Base camp
- FractaLog
- Rich stigmergy
- Mission
- Agent lifecycle

## Spirit of Synthesis

**Type:** philosophical / cognitive / methodological concept  
**Scope:** Global  
**Status:** Seed

**Short definition:**  
The motive and means connecting The Possible to the actualized. It is the operative faculty that links dispersed traces, compares actualizations, detects latent forms, designs tests, and helps a possible cross the threshold into reality. It can be read as a structural, non-theological reinterpretation of the mediating role classically associated with spirit.

**Parent concepts:**
- The Possible
- Potentics

**Related concepts:**
- Synthesis
- Theory candidate
- Rational Odyssey
- Machine à explorer
