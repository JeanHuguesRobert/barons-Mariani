# Démocratisation démocratique des réseaux sociaux

## Objet

Ce document propose une première doctrine de contrôle démocratique des réseaux sociaux.

Il part d’un constat simple : les réseaux sociaux structurent une part croissante du réel politique perçu.

Ils décident, souvent de façon opaque :

- ce qui est rendu visible ;
- ce qui est rendu invisible ;
- ce qui est amplifié ;
- ce qui est ralenti ;
- ce qui est démonétisé ;
- ce qui est déréférencé ;
- ce qui est présenté comme fiable ;
- ce qui est présenté comme suspect.

Ces décisions ne sont pas seulement techniques. Elles sont politiques par leurs effets.

## Principe central

```text
Pas de plateforme souveraine sans contrôle citoyen.
```

Ou, plus précisément :

```text
Ce qu’une plateforme rend visible ou invisible est un acte politique. Cet acte doit devenir traçable, contestable et contrôlable.
```

## Ne pas confondre démocratisation et capture

La démocratisation d’un réseau social ne consiste pas à remplacer un propriétaire privé par :

- l’État ;
- un parti ;
- une agence de vérité ;
- une fondation opaque ;
- un comité d’experts non mandaté ;
- une ONG devenue clergé informationnel ;
- une IA ou un essaim d’agents IA.

Elle consiste à rendre la plateforme contrôlable par des procédures publiques, contradictoires, auditables et démocratiques.

## Critères de classement démocratique

L’enjeu démocratique d’une plateforme dépend notamment de :

1. sa portée massive ;
2. son rôle dans l’information politique ;
3. son pouvoir d’amplification algorithmique ;
4. son pouvoir d’invisibilisation ;
5. son rôle dans les campagnes électorales ;
6. son influence sur les jeunes ;
7. sa capacité à fragmenter la réalité commune ;
8. la difficulté du recours ;
9. son degré d’opacité ;
10. sa dépendance à des infrastructures propriétaires.

## Contrôles démocratiques souhaitables

### 1. Contrôle de l’amplification

Il faut savoir :

- qui est poussé ;
- qui est enterré ;
- pourquoi ;
- selon quels critères ;
- avec quels effets mesurables ;
- avec quels recours.

La suppression des contenus n’est qu’une partie du problème. L’amplification algorithmique est souvent plus déterminante.

### 2. Contrôle de la publicité politique

Chaque citoyen doit pouvoir savoir :

- qui paie ;
- pour quel message ;
- auprès de quel public ;
- avec quelles variantes ;
- pendant quelle période ;
- avec quelle dépense ;
- avec quel intermédiaire.

### 3. Contrôle des algorithmes

Il ne s’agit pas nécessairement de publier tout le code brut.

Il faut surtout :

- des audits indépendants ;
- des parcours d’observation reproductibles ;
- des API de recherche complètes ;
- des simulations ;
- des indicateurs publics ;
- des comparaisons entre fil chronologique, fil personnalisé et fil institutionnel ;
- des tests adversariaux.

### 4. Contrôle des recours

Toute décision significative doit pouvoir être contestée :

- suppression ;
- restriction ;
- démonétisation ;
- suspension ;
- déréférencement ;
- invisibilisation ;
- limitation de portée ;
- refus de publicité.

Un recours démocratique doit être clair, daté, motivé, traçable et contradictoire.

### 5. Contrôle par interopérabilité

Le citoyen doit pouvoir partir sans mourir socialement.

La démocratisation passe donc par :

- export des données ;
- export du graphe social ;
- portabilité des communautés ;
- clients alternatifs ;
- protocoles ouverts ;
- fédération ;
- publication croisée ;
- archives portables.

### 6. Contrôle des IA

Les plateformes sociales nourrissent les IA.
Les IA produisent ensuite du contenu pour les plateformes.
La boucle devient politique.

Il faut donc distinguer :

- contenu humain ;
- contenu IA ;
- contenu humain assisté par IA ;
- bot ;
- amplification automatisée ;
- campagne coordonnée ;
- deepfake ;
- publication authentique mais augmentée.

## Formes discutables qui reviendront

Les propositions suivantes reviendront régulièrement. Elles peuvent parfois répondre à un problème réel, mais elles sont dangereuses si elles ne sont pas strictement encadrées.

### Censure administrative rapide

Risque : remplacer la capture privée par capture d’État.

### Ministère de la Vérité

Risque : créer une autorité officielle du vrai, facilement capturable.

### Obligation d’identité réelle

Risque : supprimer l’anonymat légitime, fragiliser opposants, lanceurs d’alerte, victimes, minorités.

### Affaiblissement du chiffrement

Risque : transformer la lutte contre la désinformation ou la criminalité en surveillance généralisée.

### Certification officielle des sources fiables

Risque : accorder un monopole réputationnel à des acteurs déjà dominants.

### Suppression préventive automatisée

Risque : déléguer à des filtres opaques une fonction politique majeure.

### Vente forcée ou nationalisation de plateforme

Risque : présenter une opération géopolitique ou capitalistique comme démocratisation.

### Partenariats opaques entre États, plateformes, ONG et fact-checkers

Risque : clergé informationnel sans mandat populaire.

## Doctrine par plateforme

### Facebook

Contrôler : groupes, pages, publicités politiques, visibilité locale, restrictions.

Garde-fou : ne pas transformer la lutte contre la désinformation en suppression des oppositions locales.

### YouTube

Contrôler : recommandations, démonétisation, déréférencement, recherche vidéo.

Garde-fou : ne pas imposer une hiérarchie officielle des médias agréés.

### TikTok

Contrôler : viralité courte, influence sur les jeunes, campagnes coordonnées, contenus synthétiques.

Garde-fou : ne pas réduire le problème à une question géopolitique. La machine attentionnelle est le cœur du sujet.

### Instagram / Reels / Threads

Contrôler : influenceurs, sponsorisation, visibilité image, restrictions de comptes militants.

Garde-fou : ne pas installer une police morale de l’image.

### WhatsApp / Channels

Contrôler : canaux publics, diffusion de masse, viralité.

Garde-fou : préserver absolument la correspondance privée et le chiffrement.

### Telegram

Contrôler : canaux publics, campagnes coordonnées, infractions manifestes.

Garde-fou : ne pas confondre opposition, anonymat et criminalité.

### Reddit

Contrôler : capture de modération, règles locales, décisions administratives.

Garde-fou : qui modère les modérateurs ?

### LinkedIn

Contrôler : réputation professionnelle, biais de recrutement, invisibilisation de critiques économiques ou sociales.

Garde-fou : ne pas créer un score social professionnel.

### Discord

Contrôler : grands serveurs publics, gouvernance communautaire, protection des mineurs.

Garde-fou : ne pas assimiler communauté fermée et complot.

### Fediverse / Mastodon / Bluesky

Contrôler : gouvernance des instances, défédération, listes de modération, clients dominants.

Garde-fou : ne pas confondre décentralisation technique et démocratie réelle.

## Ligne de crête

Il faut refuser deux illusions symétriques.

Première illusion :

```text
La liberté consiste à laisser les propriétaires privés gouverner invisiblement la conversation publique.
```

Deuxième illusion :

```text
La démocratie consiste à confier à l’État ou à ses auxiliaires le pouvoir de dire le vrai.
```

La voie démocratique exige autre chose :

```text
procédure, contradiction, traçabilité, recours, audit, interopérabilité, responsabilité.
```

## Formule finale

```text
Ne pas capturer les plateformes. Rendre leur capture impossible.
```

Ou :

```text
Empêcher qu’un acteur privé, public ou para-public puisse acheter le match cognitif.
```
