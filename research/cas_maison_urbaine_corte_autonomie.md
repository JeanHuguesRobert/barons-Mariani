# Maison individuelle urbaine à Corte — cas pilote anonymisé

## 1. Statut du document

Ce document concentre les données connues, hypothèses, décisions provisoires et points à vérifier concernant un cas pilote de maison individuelle urbaine à Corte étudié dans le cadre de l’École Mariani de l’Autonomie.

Le cas doit rester anonymisé dans le corpus public tant que la personne concernée ne s’est pas engagée explicitement dans la démarche ou n’a pas donné son accord pour être nommée.

Nom public recommandé :

> Maison individuelle urbaine à Corte — cas pilote anonymisé.

Nom interne possible, à ne pas publier sans accord :

> Cas Leonelli.

---

## 2. Thèse du cas

Le cas ne vise pas seulement à réduire une facture EDF.

Il vise à tester, dans une maison familiale urbaine réelle, la possibilité d’une coupure totale du lien domestique à EDF, tout en gardant une réversibilité pratique.

Formule :

> Couper EDF, ici, n’est pas une fuite hors du système. C’est une expérimentation réversible de sortie du flux permanent.

L’hypothèse forte est la suivante :

> Une maison individuelle urbaine ordinaire, chauffée principalement au bois, disposant d’une toiture solaire exploitable, d’un garage, d’une compétence électrique familiale et d’un accompagnement par un coach d’autonomie, peut passer d’une logique de circuit permanent à une logique de packets d’énergie.

Le cas est un saut volontaire, mais non irresponsable :

- le retour à EDF reste possible si l’expérience échoue ;
- le risque technique est borné ;
- le risque principal est l’acceptabilité familiale ;
- la méthode doit donc rendre le saut mesurable, réversible, documenté et transmissible.

## 3. Cadre pédagogique

Ce cas relève de la Mariani School of Autonomy.

Il ne s’agit pas d’un chantier classique d’installateur, ni d’un simple achat d’équipements.

Il s’agit d’un cas-école d’autonomie accompagnée.

Rôles :

| Acteur | Rôle |
|---|---|
| Coach d’autonomie | Structuration, méthode, arbitrages, documentation, transmission |
| Habitant principal | Apprenant-décideur, responsable du foyer, porteur du cas concret |
| Épouse / famille | Parties prenantes centrales de l’acceptabilité |
| Père et oncle, électriciens à la retraite | Tiers de confiance techniques, audit pratique, mémoire métier, soutien familial |
| IA | Aide à la mesure, reformulation, simulation, comparaison, documentation |

Formule :

> Le coach ne remplace pas l’électricien ; il rend l’habitant capable de comprendre ce que l’électricien fait, pourquoi il le fait, et dans quelles limites.

Le père et l’oncle ne sont pas seulement une aide technique. Ils apportent une capacité familiale interne :

- audit du tableau ;
- repérage des circuits ;
- lecture réaliste des contraintes ;
- dialogue éclairé avec un éventuel installateur ;
- contrôle de cohérence ;
- transmission intergénérationnelle d’un savoir pratique.

Point de prudence :

> Les anciens électriciens sécurisent l’intelligence du chantier ; ils ne doivent pas devenir les porteurs involontaires du risque juridique ou assurantiel.

Les points sensibles doivent rester traités avec rigueur : photovoltaïque, batterie stationnaire, onduleur hybride, protections, entrée générateur, conformité, assurance.

## 5. Points à vérifier en priorité

### 5.1 Factures et contrat EDF

À obtenir :

- consommation annuelle en kWh ;
- consommation par mois ;
- puissance souscrite ;
- option tarifaire ;
- montant abonnement ;
- montant total annuel ;
- historique sur 12 à 24 mois si possible ;
- existence éventuelle d’un compteur Linky accessible aux données fines.

Questions :

- quelle est la consommation de base ?
- quelle est la pointe de puissance appelée ?
- quelle est la saisonnalité ?
- quelle part est due à l’eau chaude ?
- quelle part est due à la climatisation ?
- quel est le talon nocturne ?

### 5.2 Électricité existante

À documenter :

- photos du tableau électrique ;
- nombre de circuits ;
- circuits spécialisés ;
- état des différentiels ;
- terre ;
- sections de câbles ;
- grosses charges ;
- possibilités de séparation des circuits critiques ;
- emplacement possible de l’onduleur et des batteries ;
- cheminement vers toiture / garage ;
- ventilation réelle du garage ;
- emplacement extérieur possible pour générateur.

### 5.3 Toiture et solaire

À vérifier :

- surface exploitable réelle ;
- orientation ;
- pente ;
- ombres portées ;
- état de la couverture ;
- facilité de pose ;
- contraintes urbanistiques éventuelles ;
- possibilité de 3, 6, puis 9 kWc ;
- priorité : autoconsommation autonome, non revente.

### 5.4 Véhicules

À vérifier :

- modèle exact de la Peugeot 206 ;
- année ;
- carburant ;
- état contrôle technique ;
- kilométrage réel annuel ;
- coût assurance ;
- coût entretien ;
- consommation réelle ;
- rôle exact du 4x4 de l’épouse ;
- voyages familiaux typiques ;
- degré d’opposition de l’épouse à l’électrique pure ;
- acceptabilité d’un hybride rechargeable.

## 6. Objectif technique

Objectif ambitieux :

> Couper totalement EDF pour le fonctionnement normal de la maison.

Mais ce saut doit rester réversible.

Architecture logique :

```text
maison familiale urbaine
→ solaire toiture
→ batterie stationnaire garage
→ délestage intelligent
→ chauffage bois
→ véhicule rechargeable comme packet mobile
→ réseau public de recharge comme backup externe
→ générateur essence comme parachute ultime
→ retour EDF possible si échec
```

Le but n’est pas l’autarcie idéologique.

Le but est une autonomie de capacité :

- comprendre ;
- mesurer ;
- réduire ;
- stocker ;
- prioriser ;
- router ;
- documenter ;
- tester ;
- corriger ;
- transmettre.

---

## 7. Passage du circuit au packet

Dans le régime EDF classique :

```text
prise = accès implicite à une puissance permanente
```

Dans le régime autonome :

```text
prise = droit conditionnel à consommer un packet d’énergie
```

La maison ne doit pas nécessairement devenir radicalement en courant continu. Contrairement au cas Mariani Village, ici l’objectif est un retrofit prudent.

Architecture souhaitée :

```text
extérieur utilisateur : maison familière, majoritairement 230 V AC
intérieur système : énergie mesurée, stockée, priorisée, routée
```

Formule :

> Garder la maison familière, mais remplacer l’évidence EDF par une grammaire locale des packets d’énergie.

---

## 8. Délestage intelligent

Le délestage intelligent est central.

Il repose sur :

- prises télécommandables avec mesure de courant ;
- modules tableau / contacteurs pour charges fortes ;
- Home Assistant ou système équivalent ;
- mesure de la batterie ;
- mesure de la production solaire ;
- règles simples de priorité ;
- scénarios de crise ;
- journalisation ;
- possibilité de reprise manuelle.

Deux formes de délestage :

| Type | Objectif |
|---|---|
| Délestage de puissance | éviter de dépasser la puissance de l’onduleur |
| Délestage d’énergie | préserver les kWh disponibles dans la batterie |

Classification provisoire des charges :

| Niveau | Type | Exemples | Règle |
|---|---|---|---|
| 0 | critiques | frigo, congélateur, box, éclairage minimal, capteurs | jamais coupées sauf crise extrême |
| 1 | importantes mais pilotables | chauffe-eau thermodynamique, informatique utile, ventilation | autorisées selon batterie / soleil |
| 2 | différables | lave-linge, lave-vaisselle, recharge outils, recharge vélo | lancées si conditions bonnes |
| 3 | confort | climatisation, four long, usages non essentiels | suspendues en crise |

Règle :

> Une batterie mal pilotée doit être grosse. Une batterie bien pilotée peut être plus petite.

---

## 9. Batterie stationnaire

Dimensionnement provisoire avant factures EDF :

| Capacité utile | Lecture |
|---:|---|
| 10 kWh | démonstrateur / tampon, trop faible pour coupure EDF assumée |
| 15 kWh | minimum sérieux, mais anxiogène |
| 20 kWh | minimum acceptable pour coupure EDF encadrée |
| 25 kWh | cible rationnelle |
| 30 kWh | tranquillité familiale forte |
| 40 kWh+ | probablement excessif avant mesures, sauf exigence particulière |

Hypothèse recommandée :

```text
cible : 25 kWh utiles
architecture : modulaire
extension possible : 30 kWh utiles
phase initiale si trésorerie limitée : 10 à 15 kWh utiles
```

La batterie garage doit couvrir :

- le cycle jour / nuit ;
- les passages nuageux ;
- un à trois jours sobres ;
- les charges critiques sur plusieurs jours ;
- le temps de décider d’activer un backup.

Formule :

> La batterie stationnaire n’achète pas seulement des kWh. Elle achète du temps décisionnel.

---

## 10. Véhicule rechargeable comme packet mobile

Le remplacement possible de la Peugeot 206 par un hybride rechargeable doit être étudié non seulement comme choix automobile, mais comme pièce du système énergétique.

Véhicule envisagé à vérifier :

- BYD Dolphin G DM-i ou modèle proche ;
- hybride rechargeable ;
- batterie autour de 18 kWh selon version ;
- V2L possible selon finition ;
- mensualisation possible type LLD / LOA.

Rôle du véhicule :

```text
voiture locale sobre
+ batterie mobile
+ interface avec bornes publiques
+ backup de crise
+ solution de voyage longue distance
+ compromis familial face au refus de l’électrique pure
```

Le véhicule n’est pas la batterie principale.

Il est un packet mobile.

---

## 11. Réseau de recharge comme backup

Point conceptuel central :

> Le réseau public de recharge devient le backup énergétique externe de la maison autonome.

Schéma :

```text
mauvais temps prolongé
→ batterie garage basse
→ confort réduit
→ décision humaine
→ départ vers borne de recharge
→ recharge rapide du véhicule
→ retour maison avec packet d’énergie
→ recharge batterie garage ou alimentation circuit prioritaire
```

Cette logique remplace partiellement :

- le groupe électrogène ;
- le raccord EDF de secours ;
- la livraison d’énergie ;
- le surdimensionnement coûteux de batterie.

Formule :

> La borne de recharge devient la station-service du micro-réseau domestique.

---

## 12. Générateur essence comme parachute psychologique

Un petit générateur essence peut être envisagé comme ultime assurance, principalement pour rassurer la famille.

Règle de sécurité absolue :

> Jamais dans le garage, même ventilé. Fonctionnement uniquement dehors, loin des ouvertures et entrées d’air.

Rôle :

- rassurer ;
- éviter de surdimensionner ;
- couvrir un scénario noir ;
- permettre d’accepter le saut EDF ;
- servir de parachute, non de pilier.

Puissance indicative :

```text
1 à 2 kW : probablement suffisant pour secours sobre
3 kW+ : souvent excessif, plus bruyant, plus lourd, plus consommateur
```

Formule :

> On ne construit pas l’avion autour du parachute, mais sa présence peut permettre à la famille d’accepter le décollage.

---

## 13. Deuxième véhicule familial

Le foyer semble disposer d’un second véhicule, probablement un 4x4 / truck Toyota utilisé notamment pour aller dans la nature.

Cette donnée change l’analyse.

Le véhicule rechargeable de l’habitant principal n’a pas besoin de tout faire.

Répartition possible :

| Véhicule | Rôle |
|---|---|
| Voiture rechargeable de l’habitant principal | trajets locaux, backup énergétique, interface borne, packet mobile |
| 4x4 de l’épouse | nature, chemins, sécurité psychologique, thermique longue distance |

Cela rend la transition plus acceptable.

Formule :

> Le véhicule d’Antoine peut devenir l’organe énergétique mobile de la maison précisément parce que le 4x4 conserve la fonction psychologique et pratique du véhicule qui passe partout.

## 14. Financement à trésorerie minimale

Le cas ne doit pas être financé comme un achat complet immédiat.

Il doit être financé par paliers.

Principe :

> Ne pas acheter l’autonomie. L’acquérir progressivement.

### 14.1 Paliers

| Palier | Contenu | Cash estimé | Décision |
|---|---|---:|---|
| 0 | Audit + mesure | 300–800 € | immédiat |
| 1 | Délestage intelligent léger | 800–2 000 € | rapide |
| 2 | Batterie initiale 8–12 kWh utiles | 5 000–10 000 € | après mesures |
| 3 | PV initial 3–6 kWc | 7 000–15 000 € | après simulation toiture |
| 4 | Extension batterie vers 20–25 kWh | 5 000–12 000 € | selon besoin confirmé |
| 5 | Véhicule rechargeable | mensualisé | différé |
| 6 | Générateur secours | 400–1 500 € | si utile psychologiquement |

### 14.2 Priorité financière

Ordre recommandé :

```text
1. mesurer
2. délester
3. documenter
4. installer petit mais extensible
5. différer la voiture
6. ajouter des kWh seulement si les mesures les justifient
```

Formule :

> Acheter d’abord de la connaissance et du contrôle, puis seulement ensuite des kWh.

## 15. Budget mensuel véhicule rechargeable

À vérifier selon offre réelle disponible.

Ordre de grandeur discuté :

```text
BYD hybride rechargeable en LLD / LOA : environ 250 à 300 €/mois hors assurance
budget complet avec assurance et énergie : environ 350 à 420 €/mois
```

Ce poste ne doit pas être justifié par l’économie de carburant seule, car le kilométrage annuel estimé est faible.

Si l’habitant principal roule environ 7 000 km/an, la valeur principale du véhicule rechargeable est capacitaire :

- backup domestique ;
- interface borne ;
- compromis familial ;
- réserve mobile ;
- véhicule local sobre.

---

## 16. Gain financier provisoire

Avant factures EDF, ordre de grandeur :

| Source de gain | Gain annuel plausible |
|---|---:|
| suppression ou forte réduction EDF | 800 à 1 400 € |
| délestage / sobriété / mesure | 100 à 300 € |
| carburant évité à 7 000 km/an | 300 à 600 € |
| gain total financier | 1 200 à 2 300 €/an |

Conclusion provisoire :

> Le cas ne doit pas être vendu comme économie rapide. Il doit être présenté comme autonomie de capacité, résilience familiale, démonstrateur pédagogique et réduction progressive d’une dépendance systémique.

## 17. Protocole expérimental proposé

### Phase A — Préparation

- photos tableau ;
- factures EDF ;
- inventaire des appareils ;
- relevé véhicules ;
- mesure toiture ;
- cartographie des circuits ;
- discussion familiale.

### Phase B — Mesure sans modification lourde

Durée recommandée : 30 jours.

- prises mesurantes ;
- suivi talon ;
- profils gros appareils ;
- consommation par jour ;
- puissance maximale appelée ;
- journal des usages.

### Phase C — Simulation sans EDF

Durée recommandée : 30 jours.

EDF reste techniquement disponible, mais la maison fonctionne selon les règles du futur système autonome.

Objectifs :

- tester les règles ;
- révéler les tensions ;
- former le foyer ;
- identifier les charges problématiques ;
- évaluer l’acceptabilité familiale.

### Phase D — Test crise volontaire

Durée : 3 à 7 jours.

Scénario :

- restriction volontaire de puissance ;
- batteries simulées basses ;
- chauffe-eau différé ;
- confort réduit ;
- activation éventuelle voiture / borne / générateur.

### Phase E — Décision

Décider :

- coupure EDF réelle ;
- report ;
- extension batterie ;
- installation PV plus forte ;
- maintien d’un mode hybride temporaire.

## 18. Formules structurantes

- Le cas concret est le manuel vivant.
- Couper EDF n’est pas tomber dans le vide ; c’est tester un saut réversible.
- Le foyer cesse d’être alimenté par défaut ; il apprend à s’alimenter par décision.
- Une batterie mal pilotée doit être grosse. Une batterie bien pilotée peut être plus petite.
- La borne de recharge devient la station-service du micro-réseau domestique.
- Le générateur n’est pas le retour du vieux monde thermique ; c’est le parachute.
- L’autonomie n’est pas l’absence de tout secours ; c’est la capacité à organiser soi-même ses secours.
- Garder la maison familière, mais remplacer l’évidence EDF par une grammaire locale des packets d’énergie.
